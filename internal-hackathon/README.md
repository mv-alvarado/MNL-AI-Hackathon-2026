# Internal Hackathon Share Pack

This folder is organized for easy sharing with participants before and during the internal hackathon.

## Recommended order

1. Start with `setup/SETUP-GUIDE.md`
2. Review the pre-read content in `materials/`
3. Explore reusable prompts and sample assets in `examples/`

## Folder structure

### `setup/`
- `SETUP-GUIDE.md` — participant setup instructions for IDE, Cline, LiteLLM, GitHub, and Git

### `materials/`
- `AI-FUNDAMENTALS.md` — pre-read learning material
### `examples/`
- `sample-prompts.md` — tested prompt examples that participants can adapt

### Other supporting folders
- `images/` — screenshots referenced by setup documentation
- `use-cases/` — scenario references for workshop activities

## Sharing guidance

- Share the entire `internal-hackathon` folder so links to `images/` continue working.
- If sending a compressed package, zip this folder as-is without changing subfolder names.
- If participants only need one starting point, point them to `setup/SETUP-GUIDE.md`.
